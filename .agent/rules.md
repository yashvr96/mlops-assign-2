# Antigravity Agent Rules

## Python Environment
- **Virtual Environment**: Always use `venv`.
- **Python Version**: 3.12 (specifically 3.12.10 if available).
- **Package Manager**: Use `pip` inside `venv`.

## Terminal
- **Shell**: PowerShell (Windows).
- **Command Separator**: Use `;` instead of `&&`.

## Server
- **Tool**: `uvicorn` (or similar depending on use-case).
